## About the course

TT905 covers the fundamentals of web development, from static page structure to dynamic, responsive interfaces. Topics include semantic HTML5, CSS3 layout techniques, JavaScript interactivity, and web publishing practices.

## Assignments

### Assignment 1 — RetroGames website
A multi-page static website about classic video game consoles (PS2, PS3, Xbox 360, and Nintendo Wii) and their most iconic titles.

**Key concepts practiced:**
- Semantic HTML5 tags (`header`, `nav`, `section`, `footer`, `table`)
- CSS3 custom properties, Grid, Flexbox, and `@keyframes` animations
- Responsive design with media queries (desktop and mobile breakpoints)
- Sticky header, CRT scanline effect, and per-page accent color theming
- Minimal JavaScript for a mobile hamburger menu

→ See [`trabalho1/`](./trabalho1) for the full source and report.
